# Инструкция по настройке Telegram-бота продавца

## Что мы настраиваем:
1. Создаём Telegram-бота
2. Получаем ключ Claude (искусственный интеллект)
3. Подключаем Google Таблицу с прайсом
4. Запускаем бота в облаке (Railway)

---

## ШАГ 1: Создать Telegram-бота

1. Откройте Telegram, найдите бота **@BotFather**
2. Напишите ему: `/newbot`
3. Придумайте название бота (например: `Менеджер магазина`)
4. Придумайте username бота — должен заканчиваться на `bot` (например: `moy_magazin_bot`)
5. BotFather пришлёт вам **токен** — длинная строка вида `1234567890:ABCdef...`
6. **Сохраните этот токен** — он нужен в Шаге 5

---

## ШАГ 2: Узнать свой Telegram ID

1. Найдите в Telegram бота **@userinfobot**
2. Напишите ему `/start`
3. Он пришлёт ваш ID — число вида `123456789`
4. **Сохраните этот ID** — это нужно чтобы бот присылал вам уведомления о заказах

---

## ШАГ 3: Получить ключ Claude (AI)

1. Откройте сайт: **console.anthropic.com**
2. Зарегистрируйтесь (нужна карта для пополнения, минимум $5)
3. Перейдите в раздел **API Keys**
4. Нажмите **Create Key**
5. Скопируйте ключ — начинается с `sk-ant-...`
6. **Сохраните этот ключ**

---

## ШАГ 4: Настроить Google Таблицу с товарами

### 4.1 Создайте таблицу
Создайте Google Таблицу с такими колонками в первой строке:
| Название | Описание | Цена | Наличие |
|----------|----------|------|---------|
| Товар 1  | Описание | 1000 | Есть    |
| Товар 2  | Описание | 2000 | Нет     |

### 4.2 Получите ID таблицы
В ссылке на таблицу найдите ID:
`https://docs.google.com/spreadsheets/d/`**ВОТ_ЭТО_И_ЕСТЬ_ID**`/edit`

### 4.3 Создайте сервисный аккаунт Google
1. Перейдите на **console.cloud.google.com**
2. Создайте новый проект (кнопка вверху слева)
3. Перейдите в **APIs & Services → Library**
4. Найдите и включите **Google Sheets API**
5. Перейдите в **APIs & Services → Credentials**
6. Нажмите **Create Credentials → Service Account**
7. Заполните имя, нажмите **Create and Continue**, потом **Done**
8. Нажмите на созданный аккаунт → вкладка **Keys** → **Add Key → JSON**
9. Скачается файл JSON — **переименуйте его в `google_credentials.json`**
10. Положите этот файл в папку `sales-bot`

### 4.4 Дайте доступ к таблице
1. Откройте скачанный JSON файл (блокнотом)
2. Найдите строку `"client_email"` — скопируйте email
3. Откройте вашу Google Таблицу
4. Нажмите **Поделиться** (кнопка вверху справа)
5. Вставьте email из шага 2, выберите роль **Редактор**, нажмите **Отправить**

---

## ШАГ 5: Создать файл с настройками

1. В папке `sales-bot` найдите файл `.env.example`
2. Создайте его копию и назовите `.env`
3. Откройте `.env` и заполните:

```
TELEGRAM_BOT_TOKEN=токен_из_шага_1
ANTHROPIC_API_KEY=ключ_из_шага_3
MANAGER_CHAT_ID=ваш_id_из_шага_2
GOOGLE_SHEET_ID=id_таблицы_из_шага_4
SHOP_NAME=Название вашего магазина
SHOP_DESCRIPTION=Краткое описание магазина и условий доставки
```

---

## ШАГ 6: Запуск локально (для проверки)

Если у вас установлен Python:
```bash
cd sales-bot
pip install -r requirements.txt
python main.py
```

Напишите вашему боту в Telegram — он должен ответить!

---

## ШАГ 7: Запуск в облаке Railway (постоянная работа)

1. Зарегистрируйтесь на **railway.app** (через GitHub)
2. Нажмите **New Project → Deploy from GitHub repo**
3. Загрузите код в GitHub (или используйте **New Project → Empty Project**)
4. В настройках проекта найдите вкладку **Variables**
5. Добавьте все переменные из файла `.env`
6. Также добавьте содержимое файла `google_credentials.json` как переменную:
   - Имя: `GOOGLE_CREDENTIALS_JSON`
   - Значение: всё содержимое JSON файла
7. Бот запустится автоматически

## Команды бота

| Команда | Для кого | Что делает |
|---------|----------|-----------|
| `/start` | Клиенты | Приветствие |
| `/orders` | Только вы (менеджер) | Список последних заказов |
